CSS

CSS는 "Cascading Style Sheets"의 약자로, 웹 페이지의 디자인과 레이아웃을 정의하는 스타일 시트 언어입니다. HTML로 작성된 웹 페이지의 내용과 구조는 CSS를 사용하여 시각적인 표현을 추가하고 개선할 수 있습니다. CSS는 HTML 요소에 적용되는 스타일과 속성을 지정합니다. 예를 들어, 텍스트의 색상, 글꼴, 크기, 배경색, 여백 등을 CSS를 사용하여 설정할 수 있습니다. 또한, 웹 페이지의 레이아웃을 지정하는데도 사용됩니다. 이를 통해 웹 개발자는 웹 페이지의 디자인을 일관되게 유지하고, 여러 페이지에 대한 스타일을 중앙에서 관리할 수 있습니다. CSS는 선택자(Selector)와 선언부(Declaration)로 이루어져 있습니다. 선택자는 스타일을 적용할 HTML 요소를 선택하는 역할을 하며, 선언부는 선택된 요소에 적용할 스타일과 속성을 정의합니다. 이러한 규칙들은 CSS 파일에 작성되며, 웹 페이지의 HTML 파일에서 요소를 사용하여 CSS 파일을 참조하게 됩니다. CSS는 웹 페이지를 더 멋지고 사용자 친화적으로 만드는데 큰 역할을 합니다. 웹 디자인의 외관과 레이아웃을 정의하여 사용자 경험을 향상시키고, 웹 페이지의 가독성과 접근성을 향상시킬 수 있습니다.
